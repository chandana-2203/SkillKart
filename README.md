## 🌟 Project Overview

SkillKart is a modern learning marketplace that enables users to browse courses, manage a shopping cart, apply discount coupons, and view real-time checkout calculations.

The application was developed as part of a software engineering case study to demonstrate frontend development skills, responsive design, validation handling, and AI-assisted development practices.

## 📋 Assumptions Made During Development

- Products are predefined.
- Coupon codes are fixed.
- User authentication is outside the scope.
- Payment integration is not required.
- Database integration is not required.
- Local storage is sufficient for maintaining cart data.

## ⚡ Challenges Encountered

One of the biggest challenges during development was understanding and customizing AI-generated code. While AI tools accelerated development, I spent time reviewing the generated components, understanding the project structure, and making modifications to ensure the application matched the assignment requirements.

Another challenge was deciding the scope of the project. Initially, I considered adding additional features such as database integration and multiple course categories, but I realized these were beyond the assignment requirements. I focused on building a clean, user-friendly application that effectively demonstrates cart management, discount calculation, and responsive design.

I also faced challenges while setting up the development environment, configuring GitHub, and deploying the application. These issues were resolved through debugging, documentation, and iterative testing, which improved my understanding of the overall development workflow..
